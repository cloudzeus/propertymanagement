# Pages CMS — hybrid, translatable (Sub-project C) — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development. Steps use checkbox (`- [ ]`).

**Goal:** All public pages (Pricing, Services[new], FAQ, Contact, legal) become CMS-managed, bilingual (el/en), SEO-complete (per-page metadata + page-type JSON-LD), reusing the A/B foundation.

**Architecture:** Additive `i18n Json` columns on `CMSPage`/`PricingTier`/`FAQ` (legacy columns stay as fallback). Accessors read per-locale via `pickLocale`. Each public page: `generateMetadata` from `PageSeo[slug]` + `<JsonLd>` page-type schema. Prose pages render `CMSPage` markdown (react-markdown + sanitize). Admin editors follow the established el/en-tab + SEO-tab pattern.

**Tech Stack:** Next.js 16.2 (`app/[locale]/*`), Prisma 7 (diff+deploy; DB scripts `npx tsx --env-file=.env`), next-intl (`getLocale`), react-markdown + rehype-sanitize, Vitest. Foundation: `lib/i18n/translatable.ts`, `lib/seo/*` (buildMetadata, schema builders, JsonLd, getPageSeo), `lib/cms/site-settings.ts`.

**Verified facts:**
- Stubs to build: `app/[locale]/{faq,contact,privacy,terms,cookie-policy}/page.tsx`. `pricing/page.tsx` works (PricingTier, EN-hardcoded, `dynamic='force-dynamic'`).
- `CMSPage` (slug/title/content/meta/status), `PricingTier` (name/description/features[]/monthlyPrice/annualPrice/highlighted/order/published), `FAQ` (question/answer/category/order/published). `CMSPage` unused in app.
- `getPageSeo(slug)`, `buildMetadata({seo,locale,path,baseUrl,defaultOgImage?})`, schema builders `faqPageSchema/productOfferSchema/serviceSchema/contactPageSchema/breadcrumbSchema`, `<JsonLd>`. `getSiteSettings()` has `defaultOgImage`.
- `app/sitemap.ts` PATHS lacks `/services`. CMS menu group "cms" in sidebar-nav (items «Αρχική»,«Ρυθμίσεις»).
- BASE = `process.env.NEXT_PUBLIC_SITE_URL ?? "https://property.dgsmart.gr"`.

```ts
import "server-only";
import { db } from "@/lib/db";
import { pickLocale } from "@/lib/i18n/translatable";
import type { Locale } from "@/i18n";

export function localizedCmsPage(row: { title: string; content: string; i18n: any }, locale: Locale) {
  const i = row.i18n;
  return {
    title: i?.title ? pickLocale(i.title, locale) : row.title,
    body: i?.body ? pickLocale(i.body, locale) : row.content,
  };
}
export async function getCmsPage(slug: string) {
  return db.cmsPage.findUnique({ where: { slug } });
}
export async function getPricingTiers() {
  return db.pricingTier.findMany({ where: { published: true }, orderBy: { order: "asc" } });
}
export function localizedTier(t: any, locale: Locale) {
  const i = t.i18n;
  return {
    name: i?.name ? pickLocale(i.name, locale) : t.name,
    description: i?.description ? pickLocale(i.description, locale) : (t.description ?? ""),
    features: i?.features ? pickLocale(i.features, locale) : (t.features ?? []),
  };
}
export async function getFaqs() {
  return db.fAQ.findMany({ where: { published: true }, orderBy: { order: "asc" } });
}
export function localizedFaq(f: any, locale: Locale) {
  const i = f.i18n;
  return {
    question: i?.question ? pickLocale(i.question, locale) : f.question,
    answer: i?.answer ? pickLocale(i.answer, locale) : f.answer,
  };
}
```
(Confirm the Prisma delegate name for FAQ — likely `db.fAQ` given model `FAQ`; verify in lib/prisma and adjust.) Run test → pass.

## Task 3: Shared metadata helper + Pricing page (per-locale + Product schema + SEO)

- [ ] **Step 1:** `lib/seo/page-metadata.ts` — reusable `buildPageMetadata(slug, locale, path)`:
```ts
import "server-only";
import { getPageSeo } from "@/lib/cms/page-seo";
import { getSiteSettings } from "@/lib/cms/site-settings";
import { buildMetadata } from "@/lib/seo/metadata";
import { pickLocale } from "@/lib/i18n/translatable";
import type { Locale } from "@/i18n";
const BASE = process.env.NEXT_PUBLIC_SITE_URL ?? "https://property.dgsmart.gr";
export async function buildPageMetadata(slug: string, locale: Locale, path: string) {
  const [t, site] = await Promise.all([getPageSeo(slug), getSiteSettings()]);
  const seo = t ? pickLocale(t, locale) : { title: "PropertyPro", description: "" };
  return buildMetadata({ seo, locale, path, baseUrl: BASE, defaultOgImage: site.defaultOgImage ?? undefined });
}
export { BASE as SITE_BASE };
```
